    PPLG - Pengembangan Perangkat Lunak dan Gim

Tentang PPLG

PPLG (Pengembangan Perangkat Lunak dan Gim) adalah jurusan yang fokus pada pengembangan software dan game. Siswa akan belajar berbagai bahasa pemrograman, desain software, dan pengembangan game.

Kurikulum

  • Pemrograman Dasar
  • Algoritma dan Struktur Data
  • Pengembangan Web
  • Pemrograman Mobile
  • Desain dan Pengembangan Game
  • Basis Data
  • Keamanan Perangkat Lunak

Prospek Kerja

  • Software Developer
  • Web Developer
  • Mobile App Developer
  • Game Developer
  • Database Administrator
  • UI/UX Designer
  • Quality Assurance Tester
